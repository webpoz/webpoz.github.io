Setup Masternode

This are the requirements to setup a Masternode, make sure you have 1,000 PZQ, VPS server

Send the Collateral Amount

From your wallet send the required amount of 1,000 PZQ to an address you generated getnewaddress "masternode" the above address is where you will send your collateral to. After sending it wait for 15 confirmation Open your Pozoqo wallet and make sure that you have enough PZQ coins to cover the masternode collateral of 1,000 PZQ. Generate a new BLS key for your masternode. To do this, click on "Tools" > "Debug console" and enter the command "bls generate". The console will output a private and public key. Keep the private key safe as you will need it later. Get the information needed to register your masternode. This includes the following:

  • Masternode IP address and port number
  • Masternode BLS public key
  • Masternode collateral transaction ID and output index

    • Create a new receiving address for your masternode. To do this, click on "Receive" and then "Create new address". Copy the address as you will need it in a later step. Create a ProRegTx transaction. To do this, click on "Tools" > "Debug console" and enter the following command:
    protx register_prepare "masternode_IP:port" "BLS_public_key" "collateral_txid" collateral_output_index "masternode_reward_address" "masternode_operator_key" "masternode_voting_key" "operator_reward" "voting_fee"
    Replace "masternode_IP:port" with the IP address and port number of your masternode, "BLS_public_key" with the public key generated in step 2, "collateral_txid" with the transaction ID of the PZQ coins you used as collateral, "collateral_output_index" with the output index of the collateral transaction, "masternode_reward_address" with the address generated in step above, "masternode_operator_key" with the private key generated in step 2, "masternode_voting_key" with a new BLS key that will be used for voting, "operator_reward" with the amount of PZQ you want to pay yourself as a reward, and "voting_fee" with the amount of PZQ you want to charge for voting.

    protx register_prepare collateralHash collateralIndex ipAndPort ownerKeyAddr operatorPubKey votingKeyAddr operatorReward payoutAddress (feeSourceAddress)


      collateralHash: The txid of the 1000 PZQ collateral funding transaction
      collateralIndex: The output index of the 1000 PZQ funding transaction
      ipAndPort: Masternode IP address and port, in the format x.x.x.x:yyyy
      ownerKeyAddr: The new Dash address generated above for the owner/voting address
      operatorPubKey: The BLS public key generated above (or provided by your hosting service)
      votingKeyAddr: The new Dash address generated above, or the address of a delegate, used for proposal voting
      operatorReward: The percentage of the block reward allocated to the operator as payment
      payoutAddress: A new or existing Dash address to receive the ownerâ€™s masternode rewards
      feeSourceAddress: An (optional) address used to fund ProTx fee. payoutAddress will be used if not specified.

    Sign the ProRegTx transaction

    Use the code below to sign the proregtx transaction
    signmessage collateralAddress signMessage
